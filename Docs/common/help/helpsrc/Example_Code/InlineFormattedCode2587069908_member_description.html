How to inline formatted code in the documentation.
Use: \code \endcode.

Inline formatted code in place
Examples are always put into the "example" section, which is at the end of the generated documentation page for the entity. When using \code \endcode statements you can inline code where you want.

For example like this:
void inlinedCode()
{
  cout<<"This is inlined code ie. not and example!\n";
  cout<<"Se also!\n";
  HowToDocument::ExampleInline();
  HowToDocument::ExampleInclude();
}


Then you can continue writing documentation here...
