Description
Template parameter documentation.
Template parameters must be documented using the Java-style \param keyword. This might however change in a future version of DoxyS allowing for the standard way also. This convention also applies when documenting class template parameters.

This is how it looks in the source:
\param T Description of a template parameter.
\param COUNT One more template parameter
