HELP - LeftMenu
HelpConfig OptionsLeftMenu
Description  
Configuration options related to Left Menu layout.
SHOW_LEFT_MENUIf set to YES then left menu will be generated. If set to NO leftmemu and logo will not be included. This option disables all left menus in all types of pages and ignores the other SHOW_LEFT_MENU_XXXX.
Default value:
YES
SHOW_LEFT_MENU_MAINDIRIf set to YES then left menu will be generated for main page. If set to NO leftmemu and logo will not be included
Default value:
YES
SHOW_LEFT_MENU_SEARCHIf set to YES then left menu will be generated for search pages. If set to NO leftmemu and logo will not be included
Default value:
YES
SHOW_LEFT_MENU_DIRECTORYIf set to YES then left menu will be generated for search pages. If set to NO leftmemu and logo will not be included
Default value:
YES
SHOW_LEFT_MENU_DOCIf set to YES then left menu will be generated for the rest of documentation pages (not above). If set to NO leftmemu and logo will not be included
Default value:
YES
SHOW_LOGOIf set to YES and SHOW_LEFT_MENU set to yes then the logo will be displayed.
Default value:
YES
LEFT_MENU_DIRECTORIESIf set to YES directories will be shown on left menu.
Default value:
YES
LEFT_MENU_PAGESIf set to YES pages will be shown on left menu.
Default value:
YES
LEFT_MENU_CLASSESIf set to YES classes will be shown on left menu.
Default value:
YES
LEFT_MENU_STRUCTSIf set to YES classes will be shown on left menu.
Default value:
YES
LEFT_MENU_FUNCTIONSIf set to YES functions will be shown on left menu.
Default value:
NO
LEFT_MENU_MACROSIf set to YES macros will be shown on left menu.
Default value:
NO
LEFT_MENU_DEFINESIf set to YES defines will be shown on left menu.
Default value:
NO
LEFT_MENU_ATTRIBUTESIf set to YES attributes will be shown on left menu.
Default value:
NO
LEFT_MENU_TYPESIf set to YES types will be shown on left menu.
Default value:
NO
LEFT_MENU_UNITSIf set to YES units will be shown on left menu.
Default value:
YES
LEFT_MENU_STATISTICSIf set to YES statistics will be shown on left menu.
Default value:
YES
STATISTICS_LINK_ON_MAIN_PAGEIf set to YES, a link to statistics will be on the main page. If set to NO then locate the statistics under the common directory.
Default value:
YES
