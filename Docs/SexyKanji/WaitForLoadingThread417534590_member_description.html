Description
Wait for the main loading thread to receive the shutdown signal.
***********************************

Description: If the user/player exits the game while the main loading thread is running, then we have to wait for the loading thread to see the shutdown message and exit so we can shutdown without causing problems (like deleting the ResourceManger that it is inevitably using).
